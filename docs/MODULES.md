# NZT Studio — Módulos funcionales

## Dashboard
Vista principal con métricas y acceso rápido a módulos.

## Companies
Gestión de empresas analizadas. CRUD completo.

## Market Intelligence
Análisis de mercado y sectores. Datos y tendencias.

## Company Analyzer
Análisis técnico de webs de empresas. Primera fase: HTML parsing. Segunda fase: Playwright + Lighthouse.

## Opportunity Engine
Detección y gestión de oportunidades digitales vendibles.

## MVP Factory
Generación de especificaciones de MVPs a partir de oportunidades.

## Pricing Studio
Generación de presupuestos estructurados.

## Proposal Builder
Creación de propuestas comerciales completas.

## Delivery Workspace
Gestión de proyectos en entrega activa.

## Knowledge Base
Base de conocimiento interna: plantillas, prompts, referencias.
